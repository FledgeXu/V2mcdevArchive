Markdown 教程
======

基础
------------

普通的一段文本。

这是另一段话。
这句话接上一句话，不会另起一行。  
如果你一定要另起一行，上一行的结尾得有至少俩空格，就像上面这句话一样。

*斜体*

**粗体**

[链接](https://www.v2mcdev.com)

![我是图](upload://rwK0jhRBWlMz3VojcZk04ve8gJr.jpeg)

标题
------------

# H1
## H2
### H3
#### H4
##### H5
###### H6

代码
------------

行内的代码可以直接 `write like this`

也可以写成这样

```
print("Hello world")
```
